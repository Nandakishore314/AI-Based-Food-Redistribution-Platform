# AI-Based Food Redistribution Platform

An AI-powered platform that connects food donors, NGOs, and volunteers to reduce food wastage.

## Features

- Food Donation Upload
- NGO Matching
- Volunteer Delivery Assignment
- Heatmap Visualization
- Demand Prediction
- Admin Dashboard

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Python
- Flask/FastAPI

Database:
- MongoDB

Machine Learning:
- Scikit-Learn
- Pandas
- NumPy
